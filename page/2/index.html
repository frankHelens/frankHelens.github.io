webpack是一款模块加载器兼打包工具，它能把各种资源，例如JS（含JSX）、ES6、样式（含less/sass）、图片等都作为模块来使用和处理。

创建 webpack.config.js 它的作用和gulpfile.js一样就是一个配置项，设置 webpack 任务功能。

  • entry 入口文件 让webpack用哪个文件作为项目的入口
  • output 出口 让webpack把处理完成的文件放在哪里
  • module 模块 要用什么不同的模块来处理各种类型的文件
  • plugins 用于提取多个入口文件的公共脚本部分，然后生成一个 common.js 来方便多页面之间进行复用。
  • resolve 用来设置路径指向
  • watch 用监听文件有改动后执行打包

Veröffentlicht am 2017-02-11

canvas

一、什么是canvas

  • canvas 元素用于在网页上绘制图形。HTML5 的 canvas 元素使用 JavaScript 在网页上绘制2D图像。
  • 在矩形区域的画布上，控制其每一像素，JavaScript 来绘制 2D图形，逐像素进行渲染。可以通过多种方法使用canvas 元素绘制路径、矩形、圆形、字符以及添加图像。
  • 浏览器支持情况 IE9.0以前的版本不支持
  • getContext(‘2d’)方法可返回一个对象，该对象提供s了用于在画布上绘图的方法和属性。

二、文字绘制

  • font设置或返回文本内容的当前字体属性
  • textAlign 设置或返回文本内容的当前对齐方式
  • textBaseline 设置或者返回在绘制文本时使用的文本基线
  • 方法:
  • fillText(text,x,y,maxWidth): text表示文字，x、y为坐标、maxWidth可选，为文字最大宽度，防止文字溢出；填充文字
  • strokeText(text,x,y,maxWidth): text表示文字，x、y为坐标、maxWidth可选，为文字最大宽度，防止文字溢出；绘制文字

三.阴影绘制

  • shadowColor 阴影颜色
  • shadowOffsetX 阴影的左右偏移值
  • shadowOffsetY 阴影的上下偏移值
  • shadowBlur 阴影模糊度
  • save() 保存当前属性设置
  • ctx.restore() 返回上一次保存的属性值,只能是保存一次返回一次
